# 🔰 (uv)^T = v^T u^T

[線代](../../../../) ⟩ [矩陣](../../../) ⟩ [運算](../../) ⟩ [乘法](../) ⟩ $$(\mathbf{uv})^T = \mathbf{v}^T \mathbf{u}^T$$

{% hint style="success" %}
若： $$\mathbf{u}$$,  $$\mathbf{v}$$ 是[行向量](../../../row-col.md)或[列向量](../../../row-col.md)，<mark style="color:yellow;">且可作</mark>[矩陣乘法](../)，則：

* &#x20;$$(\mathbf{uv})^T = \mathbf{v}^T \mathbf{u}^T$$

:star: 注意： $$\mathbf{u}$$,  $$\mathbf{v}$$ <mark style="color:yellow;">可作</mark>[矩陣乘法](../)，有兩種狀況：

* 「$$1 × n$$  矩陣」乘以「$$n × 1$$  矩陣」，得到一個「$$1 × 1$$  矩陣」，也就是一個<mark style="color:yellow;">純數</mark>。
* 「$$m × 1$$  矩陣」乘以「$$1 × n$$  矩陣」，得到一個「$$m × n$$  矩陣」，也就是「 [.](./ "mention") 」。
{% endhint %}
