# 🔰 函數

[Desmos](../../) ⟩ [數學式](../) ⟩ 函數

{% hint style="warning" %}
Desmos 的自訂函數可以使用 $$x,y,z$$ 以外的參數，例如：

$$f(a,b,c)=a+b+c$$

* 但缺點是這些參數<mark style="color:red;">**不能**</mark>拿來當作<mark style="color:yellow;">**自由變數**</mark>:exclamation:
* 還好，當作<mark style="color:yellow;">**其他函數的參數**</mark>還是<mark style="color:green;">**可以**</mark>的。
{% endhint %}
